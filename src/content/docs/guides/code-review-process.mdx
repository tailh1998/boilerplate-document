## Follow the rule below:

1. Make sure your code pass ESLint

2. Resolve conflicts (if any)

3. Create a Merge Request, and comment the link to `fe-pr` channel with the following syntax: `[MR name] [Link] [Reviewers]`. Tag at least **2 members** to review your MR

4. After the MR has at least **2 approvals**, the **last person** who approves will merge, and **comment "Merged"** to the thread.
